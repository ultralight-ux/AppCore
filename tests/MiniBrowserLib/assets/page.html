Ultralight MiniBrowser

About

MiniBrowser can be used to test, inspect, and profile web-content using the Ultralight renderer.

Simply navigate to a URL in the address bar above to begin.

Local HTML files can be placed inside the assets folder and accessed via: 

    file:///filename.html

To inspect pages locally:

Press the wrench icon in the upper-right.

To inspect pages remotely:

Press F2 to start the remote inspector server then open a second instance of MiniBrowser and navigate to: 

    inspector://127.0.0.1:7676
